# 🛖 Murat'ın Yeri - Video Chat Platformu

Bu proje, bir "Omegle/SarmasDolas" tarzı canlı yayın ve video sohbet platformudur. 

## 🚀 Hızlı Başlangıç

### 1. Backend (Server)
- Dizine git: `cd server`
- Gereksinimleri yükle: `npm install`
- Başlat: `node index.js`
- **Önemli:** `.env` dosyasındaki Agora APP_ID ve SQL bilgilerini doldurun.

### 2. Frontend (Client)
- Dizine git: `cd client`
- Gereksinimleri yükle: `npm install`
- Başlat: `npm run dev` (Site yayına girecektir)

## ✨ Özellikler
- 🎥 **RTC Görüntülü Sohbet:** Düşük gecikmeli Agora RTC desteği.
- 💬 **Canlı Chat:** Her odaya özel anlık mesajlaşma (Socket.io).
- 🎁 **Hediye Sistemi:** Gül, Pırlanta ve Araba fırlatma imkanı.
- 🛡️ **Güvenlik:** Rate limiting, Helmet ve moderasyon (Kick) araçları.

Murat'ın Yeri artık senin, hayırlı olsun! 🚀✨
